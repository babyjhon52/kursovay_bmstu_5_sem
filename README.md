# Система анализа МРТ-снимков головного мозга

Интеллектуальная система для автоматической сегментации опухолей на медицинских изображениях с использованием глубокого обучения.

## Быстрый старт

### Установка

```bash
pip install -r requirements.txt
```

### Запуск

```bash
python main.py
```

### Учетные данные по умолчанию

* Администратор: `admin` / `admin123`
* Врач: `d` / `1`
* Исследователь: `r` / `1`

## Структура проекта

```
LGG_SEGM/
├── main.py              # Точка входа
├── src/                 # Исходный код
│   ├── core/           # Основные модули
│   └── gui/            # Интерфейс
├── tests/              # Тесты
├── data/               # Данные
└── docs/               # Документация
```

## Тестирование

```bash
pytest tests/ -v
```
